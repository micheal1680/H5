存取器属性；

  1. 属性值可以由一个或两个方法替代，这两个方法就是getter和setter；
  2. 由getter和setter定义的属性，称为“存取器属性”； 一般的只有一个值的属性称为“数据属性”；
  3. 查询存取器属性的值，用getter；拥有getter则该属性可读；
  4. 设置存取器属性的值，用setter；拥有setter则该属性可写；
  5. 定义一个简单的有存取器属性的对象：不看例子你根本不知道在说什么 
                    var myobj = {
                    //数据属性
                    myname:'yourname',
                    birthday:'1983-05-17',
                    get myage(){
                        return (new Date().getFullYear()) - new Date(this.birthday).getFullYear();
                    },
                    set myage(value){
                        this.birthday = value;
                    }
                };
                myobj.myage = '1998-08-22';
                console.log(myobj.myage);

                    var getnum = {
