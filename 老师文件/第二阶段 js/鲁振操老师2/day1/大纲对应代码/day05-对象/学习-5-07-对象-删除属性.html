delete运算符可以删除对象的属性；

            delete person1.name;
            delete person1.age;
  1. 只能删除自有属性，不能删除继承属性；
  2. delete删除成功或删除不存在的属性或没有副作用时，返回true；
  3. delete不能删除可配置性为false的属性； 
                        var x = 1;
                    delete this.x; //false
                    function my(){}
                    delete this.my;//false

                    this.a = 100; //没有var
                    delete a; //true
