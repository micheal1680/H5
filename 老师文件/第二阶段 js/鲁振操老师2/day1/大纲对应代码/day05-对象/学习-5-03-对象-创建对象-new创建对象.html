new用于创建新的对象；

  1. new后面跟一个函数表示创建对象；这里的函数是构造函数(constructor)；
  2. 用new创建几个内置对象： 
                    var obj1 = new Object();
                console.log(obj1);
                var obj2 = new Date();
                console.log(obj2);
                var obj3 = new Array();
                console.log(obj3);
