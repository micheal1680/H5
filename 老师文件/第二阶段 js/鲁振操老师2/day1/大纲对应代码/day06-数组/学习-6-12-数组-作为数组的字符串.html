可以把字符串看成数组来访问；

  1. 把字符串看成数组，只能访问其中的“元素”，不能修改； 
                        var str = 'hqyjhyn';
                    console.log(str[4]); //charAt()：指定位置的字符
                    console.log(str.charAt(4));//中括号的方式访问起来更方便
    数组的方法可以劫持过来使用在字符串上，这就很帅了； 
                        //劫持数组的方法来使用
                    var b = Array.prototype.map.call(str, function (v) {
                        return v+v;
                    });
                    console.log(str);
                    console.log(b);
