数组对象有哪些方法我们可以使用；

  1. join()： 数组中的元素转换成字符串并连接在一起； 默认是逗号(,)连接，可以指定连接符号； 
                        var students = [
                                    ['张三', 20],
                                    ['李四', 21],
                                    ['王五', 22],
                                    ['小六', 23]
                                ];
                    console.log(students.join());
                    console.log(students.join('|'));
  2. reverse()：把数组中的元素顺序反转； 
                        var students = [
                                    ['张三', 20],
                                    ['李四', 21],
                                    ['王五', 22],
                                    ['小六', 23]
                                ];
                    console.log(students.reverse().join('|'));
                    console.log(students);
  3. sort()：对数组中的元素排序并返回排序后的值； 没有参数时按字母排序，会把元素转换成字符串； 
                        var a = ['a', 'd', 'f', 'b', 'k','c'];
                    a.sort();
                    console.log(a);
    对元素是数字的数组进行排序时，是需要注意的； 
                        //下面的排序结果可能不是你想要的
                    var a = [1, 23, 9, 12, 80];
                    a.sort();
                    console.log(a);
                    //也许下面的排序才是你想要的
                    var a = [1, 23, 9, 12, 80];
                    a.sort(function (a,b) {
                        /* 从小到大，a在前面，返回负值 */
                        // return a-b;
                        /*从大到小*/
                        return b-a;
                    });
                    console.log(a);
  4. concat()：数组连接，连接后返回一个新数组，千言万语不如代码十行： 
                        //猜一猜他们会打印什么到控制台？
                    var a = ['h', 'q', 'y', 'j'];
                    console.log(a.concat('h5'));
                    console.log(a = a.concat('javaee', '嵌入式'));
                    console.log(a.concat(['AR/VR', 'Python']));
                    console.log(a.concat(['AR','VR'], ['Python',['AI', '大数据', '运维自动化']]));
  5. slice()：截取数组，两个参数，指定开始和结束位置；
    参数支持负数，表示从后面往前面数，第一个-1, 依次-2,-3...；
                        var a = ['h', 'q', 'y', 'j'];
                    console.log(a.slice(0, 2));
                    console.log(a.slice(2, 2));//指定同一个位置会怎样？
                    console.log(a.slice(2));
                    console.log(a.slice(-2));
                    console.log(a.slice(-2, -1));
                    console.log(a.slice(-2, -3));//这种会怎样？
  6. splice()：插入、删除元素的通用方法，哎哟，这么牛逼，好怕；
    感觉插入、删除元素的其他方法可以去死了；
    会直接修改元数组；
    参数1：操作起始位置；
    参数2：操作元素个数，为空表示到结尾，为0表示插入元素，大于0表示替换元素；
    参数n：新的元素，代表要插入的元素； 
                        ar a = ['h', 'q', 'y', 'j'];
                    a.splice(2); //从下标2开始删除
                    console.log(a);
                    a.splice(2, 0, 1, 2, 3);//从下标2开始添加元素
                    console.log(a);
                    a.splice(2, 1, 'y', 'j');//从下标2开始添加元素
                    console.log(a);
                    a.splice(1, 0, ['H5', '嵌入式']);//这里插入数组元素会怎么样那？
                    console.log(a);
  7. push():在数组的结尾添加一个或多个元素，返回新数组的长度；
    pop():移除数组最后一个元素，返回移除的元素；
    unshift():在数组的开头添加一个或多个元素，返回新数组的长度；
    shift():删除数组第一个元素，并返回删除的元素；
    这四个函数都是在原始数组上进行修改
  8. toString():将数组的所有元素转换成字符串，并用逗号连接起来并输出; 让我想起来join(); 
                        var a = [
                                'h', 'q', 
                                ['H5', '嵌入式'], {x:1, y:2},'y',
                                function () {console.log('我是函数');}, 
                                'j'
                            ];
                    console.log(a.toString());
                    console.log(a.join('|'));
