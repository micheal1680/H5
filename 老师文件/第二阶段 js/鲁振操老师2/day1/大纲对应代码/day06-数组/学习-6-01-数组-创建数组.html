数组是对象的特殊表达形式；

  1. 数组的元素可以是数字、值、表达式、对象等;
    定义数组的几种方式； 
                        //中括号的方式，简单方便又好看
                    var arr1 = [1,2,3,45,8,6];
                    console.log(arr1);
                    //空数组
                    var arr2 = new Array();
                    //指定长度的数组
                    var arr3 = new Array(10);
                    //指定元素
                    var arr4 = new Array(12, 34, 5,6,7,8);
                    //数组元素支持表达式
                    var h = 10;
                    var arr5 = new Array(h+1, h+5, h+9);
                    //元素可以是其他类型
                    var arr6 = new Array({x:1, y:2}, h+5, h+9);
