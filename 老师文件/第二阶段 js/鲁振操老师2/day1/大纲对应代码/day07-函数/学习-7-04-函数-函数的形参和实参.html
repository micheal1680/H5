函数的参数非常自由；

  1. 调用函数时传递的参数，全部保存在一个类数组arguments里面，函数内部可用；
  2. 如果传递的实参少于形参个数，则多出的形参是undefined；
  3. 如果你想给参数设置默认值，也许下面的方法是一个很好的选择： 
                        function divide(a, b){
                        var a = a || 0;//常用的设置默认值的方式
                        var b = b || 1;
                        return a/b;
                    }
  4. 如果传递的实参多余形参，没毛病；
  5. 终上所述，看个小码； 
                    function divide(a, b){
                    var a = a || 0;//常用的设置默认值的方式
                    var b = b || 1;
                    return a/b;
                }
                console.log(divide(1));
                console.log(divide(3,2,1));

arguments.callee属性和函数.caller属性；

  1. callee属性指当前正在执行的函数；
  2. caller属性指当前正在执行的函数的函数；非标准，大部分浏览器支持；
  3. 严格模式下不可用，会报错；
  4. 走一波代码： 
                        function fn () {
                        console.log(arguments.callee); 
                    }
                    fn();
    再来个小代码： 
                        function a () {
                        console.log(a.caller);
                        console.log(arguments.callee.caller);
                    }
                    function b () {
                        console.log('函数b');
                        console.log(b.caller);
                        a();
                    }
                    b();
