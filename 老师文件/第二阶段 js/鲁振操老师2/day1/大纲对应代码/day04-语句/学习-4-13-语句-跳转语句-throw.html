throw用于抛出异常；

  1. 立即停止正在执行的程序，跳转至就近的逻辑异常处理程序；
  2. 反正看不懂，写段代码看看： 
                        function factorial (n) {
                        if(isNaN(n)) throw new Error('请输入数字，HOHO');
                        if(n == 1) return 1;
                        return n*factorial(n-1);
                    }
                    var n = factorial(3);
                    console.log(n);

                    var n = factorial('a05');
                    console.log(n);

                    var n = factorial(5);
                    console.log(n);
