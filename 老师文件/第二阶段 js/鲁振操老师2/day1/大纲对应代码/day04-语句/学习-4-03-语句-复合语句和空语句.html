用大括号将多条语句括起来就是复合语句

  1. 例子是万能的： 
                    {
                    var now     = new Date();
                    var year    = now.getFullYear();
                    var month   = now.getMonth() + 1;
                    var day     = now.getDate();
                    console.log('今天是'+year+'年'+month+'月'+day+'号');
                }
  2. 你也看到了，语句块的结尾不需要分号；
  3. 整齐缩进的代码看上去更安逸，更容易理解；
  4. 当需要把多条语句当做一条语句来执行时，需要用复合语句；比如循环； 
                        var age = 1;
                    while (age < 100) {
                        document.write('我今年' + age + '岁了；');
                        age++;
                    }
  5. 需要注意的是这里面的变量外面可以访问吗？ JavaScript说他没有传说中的块级作用域？ 且听下回分解！
