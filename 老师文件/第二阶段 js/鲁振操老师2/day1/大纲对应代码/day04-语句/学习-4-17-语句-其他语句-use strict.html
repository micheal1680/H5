use strict是将后续的代码解析为严格代码；

  1. 严格模式下禁止使用with；
  2. 严格模式下所有的变量都要先声明；
  3. 严格模式下调用函数的this值是undefined；下面这段小代码不加"use strict"的区别； 
                        "use strict";
                    function fn(){
                        console.log(this);
                    }
                    fn();
  4. 严格模式和普通模式有很多细微差别，记住很困难，基本原则是：写代码要严谨，不要偷懒，不要觉得当前代码可以正常运行即可；
