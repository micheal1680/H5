break用于结束一个循环

continue用于结束当次循环，并继续执行下一次循环

  1. 标签由标识符加冒号组成，如：gohere:；
  2. 标签标识符+冒号(:)+语句，就组成了标签语句；
  3. break 和 continue可以使用语句标签；
  4. 啥也不说了，看代码： 
                        var a = 0;
                    forouter_b:
                    for (var i = 0; i < 10; i++) {
                        for (var j = 0; j < 10; j++) {
                            if(j == 5){
                                break forouter_b;
                            }
                            a++;
                        }
                    }
                    console.log(a);
    体会break后面加和不加语句标签的区别； 
                        var a = 0;
                    forouter_c:
                    for (var i = 0; i < 10; i++) {
                        for (var j = 0; j < 10; j++) {
                            if(j == 5){
                                continue forouter_c;
                            }
                            a++;
                        }
                    }
                    console.log(a);
    体会continue后面加和不加语句标签的区别；
