布尔类型

  1. 布尔值指对或错、开或关、是或否;
  2. 布尔类型只有两个值：true 和 false，是保留字，小写；
  3. 布尔值通常用于控制语句中，如if、for、while等；
  4. JavaScript的任意值都可以转换为布尔值，可直接使用Boolean()进行转换， 下面的值会转换为false： 
                    undefined
                null
                0
                -0
                NaN
                ''
    其他值则转换为true，包括空数组、对象等，好神奇；
