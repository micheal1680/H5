构造函数用来创建新对象；

  1. 构造函数首字母大写；一般用于初始化一些属性值； 
                        function Range (from, to) {
                        this.from   = from;
                        this.to     = to;
                    }
  2. 构造函数的prototype属性用于定义该函数对象的原型；
    该原型会被所有的实例对象继承； 
                        Range.prototype = {
                        includes:function (a) {
                            return a >= this.from && a <=this.to;
                        },
                        foreach:function (f) {
                            for (var i = this.from; i <= this.to; i++) {
                                f(i);
                            }
                        },
                        avage:function () {
                            var n = 0;
                            for (var i = this.from; i <= this.to; i++) {
                                n += i;;
                            }
                            return (n/(this.to - this.from + 1)).toFixed(2);
                        },
                        setfrom:function (from) {
                            this.from = from;
                        },
                        setto:function (to) {
                            this.to = to;
                        }
                    };
  3. 使用关键词new调用构造函数创建一个实例对象； 
                        var nr = new Range(2, 20);
  4. 新创建的所有实例对象会继承构造函数的原型，也就是prototype属性； 
                        console.log(nr);
                    nr.setfrom(10);
                    nr.setto(15);
                    nr.foreach(console.log);
                    var avg = nr.avage();
                    console.log(avg);
  5. 此时原型对象或实例都没有constructor(构造函数)属性，需要显式指定： 
                        Range.prototype = {
                        constructor:Range,
                        ...其他属性或方法...
                    }
