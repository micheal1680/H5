总结

Boolean类型

  1. 值: true false
    1. string: "" false 非空字符串 true
    2. undefined: false
    3. object: null false 非空对象 true
    4. number: 0/-0 NaN false 非零数值 true

Number类型

  1. 整数 9 10 浮点数 1.01 2.0==>整数 2.==>整数 .9 (不建议使用)
  2. 精度问题: 解决办法: 转为整数计算 保留小数 Math.floor(x), Math.ceil(x)
  3. 其他类型转为Number类型: Number() parseInt() parseFloat() +/-/*// , ++ ,--, -/+
  4. NaN : 0 / 0 ==> NaN 特点: NaN == NaN ==> false

Object类型

  1. 对象的创建方法 var obj = {} var obj = new Object()
  2. 访问: . [] .语法无法访问数值属性, a.1 // 错误 []: 访问中括号中的值不能直接写属性名,需要改为字符串访问
  3. 包装对象
    1. 临时包装对象: 访问属性后会被销毁
    2. 显示包装对象: var s = "string"; var S = new String(s); S不会被销毁

变量作用域

  1. 全局作用域: window/this
  2. 局部作用域(函数作用域): 函数内部声明的变量(必须使用关键字var声明)
  3. 函数内部可以访问外部变量 外部不能访问内部变量(报错)

操作符

  1. 一元操作符: ++ -- + -
  2. 位操作符: ~ & | ^ << >> >>>
  3. 逻辑操作符: ! && ||
    1. !: !! 获得当前值的布尔值
    2. &&(逻辑与/短路与) : 第一个值的结果为真,返回第二个值的结果, 第一个值的结果为假,则直接返回第一个值的结果
    3. ||(逻辑或/短路或) : 第一个值的结果为真,返回第一个值的结果,第一个值的结果为假,返回第二个值的结果
  4. 乘性操作符: * / %
  5. 加性操作符: + -
