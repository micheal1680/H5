## 3.组件中自定义函数的使用

组件中可以任意自定义函数，使用this调用

```js
import React from "react";
export default class App extends React.Component {
  getView() {
    let isShow = 3;
    if (isShow === 3) {
      return <div>你是对的</div>;
    } else {
      return <div>错误答案</div>;
    }
  }
  render() {
    return this.getView();
  }
}
```
